DES 250: Digital Media Design II


Overview

This course expands on basic principles of interactivity and motion design. Emphasis is placed on developing a critical understanding of digital literacy.

Objectives + Goals

  • Design and code web pages using HTML and CSS
  • Understand and apply responsive web design practices
  • Utilize design processes and methods
  • Create visual hierarchy and contrast
  • Learn to iterate in the digital design process
  • Work with code editors
  • Explore the limits and possibilities of interactivity
  • Prepare digital files for web-based publishing
  • Design and publish pages to the web
